# Escape Estelar

Escape Estelar es un videojuego de aventura desarrollado en Python para consola como parte de la competencia PyWeekend.

La historia se desarrolla en una estación espacial que ha sido gravemente dañada por una lluvia de meteoritos. El jugador deberá explorar diferentes habitaciones, encontrar herramientas y recursos esenciales, y resolver distintos desafíos para lograr escapar antes de la destrucción total de la nave.

## Características

* Exploración de múltiples habitaciones.
* Sistema de inventario.
* Objetos interactivos.
* Narrativa basada en texto.
* Gráficos ASCII generados en consola.
* Experiencia completamente jugable desde la terminal.

## Tecnologías utilizadas

* Python
* Arte ASCII

## Objetivo del proyecto

Este proyecto fue desarrollado con el objetivo de practicar conceptos fundamentales de programación, diseño de videojuegos y resolución de problemas mediante Python.

Además de representar uno de mis primeros proyectos completos, Escape Estelar fue una experiencia importante para aprender a estructurar programas más grandes y trabajar en una aplicación de principio a fin.

## Cómo ejecutar el proyecto

1. Clonar el repositorio.
2. Abrir el proyecto en Visual Studio Code o cualquier editor compatible con Python.
3. Ejecutar el archivo principal del juego.

```bash
python main.py
```
